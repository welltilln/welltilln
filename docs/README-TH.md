### สรุปสั้นๆ

นักศึกษาบริหารธุรกิจ ม.เชียงใหม่ ที่เรียน CS ข้ามคณะเพิ่ม ภาษาอังกฤษระดับ C1 มีใบรับรอง Cybersecurity 2 ใบ เคยเป็นหัวหน้าฝ่ายผลิตงานเทศกาลดนตรีขนาดใหญ่ และเคยทำงาน Database Consultant ให้บริษัทมหาชน ตอนนี้สนใจตำแหน่ง BA/DA ที่ได้ใช้ทั้งความเข้าใจธุรกิจและทักษะเทคนิคไปด้วยกัน

---

### ผลงานของผมที่คัดมา

#### [howmanycals](https://github.com/welltilln/howmanycals)
**LINE Bot นักโภชนาการ AI**
*   **ทำอะไร:** สร้าง LINE Bot ที่ถ่ายรูปอาหารแล้วบอกแคลอรี่ได้ทันที ใช้ Gemini Vision วิเคราะห์ภาพแล้วแปลงเป็นข้อมูลโภชนาการแบบมีโครงสร้าง
*   **เทคโนโลยี:** Python, FastAPI, Google Gemini Vision API, SQLite
*   **จุดเด่น:** มีระบบนับแคลอรี่สะสมรายวัน รีเซ็ตอัตโนมัติตอนเที่ยงคืน ใช้งานได้จริงในชีวิตประจำวัน

<p align="center">
  <img src="../howmanycals_preview.png" alt="How Many Cals Bot Preview" width="100%">
</p>

#### [fastapi-line-gemini](https://github.com/welltilln/fastapi-line-gemini)
**ต้นแบบ LINE Bot + AI แบบพร้อมใช้**
*   **ทำอะไร:** ออกแบบ Starter Kit สำหรับคนที่อยากต่อ LLM เข้ากับ LINE Bot โดยไม่ต้องเริ่มจากศูนย์ จัดโครงสร้างโค้ดให้ขยายต่อได้ง่าย
*   **เทคโนโลยี:** Python, Docker, Ngrok, LINE Messaging API
*   **จุดเด่น:** รองรับ 5 ภาษาตั้งแต่แรก ทั้งเอกสารและตัว Bot เอง

#### [Yosafe](https://github.com/welltilln/yosafe)
**ระบบบันทึกและตรวจสอบสินทรัพย์**
*   **ทำอะไร:** สร้างระบบบัญชีส่วนตัวสำหรับบันทึกการเคลื่อนไหวของเงินและสินทรัพย์ ออกแบบให้ตรวจสอบย้อนหลังได้ทุกรายการ ข้อมูลต้องตรง 100%
*   **เทคโนโลยี:** SQL (PostgreSQL), Python (TUI), Bash

<p align="center">
  <img src="../yosafe_preview.png" alt="Yosafe TUI Command Center" width="100%">
</p>

#### [agent-asylum](https://github.com/welltilln/agent-asylum)
**คลังบันทึกความล้มเหลวของ AI Agent**
*   **ทำอะไร:** เก็บบันทึกกรณีที่ AI Agent ทำงานผิดพลาด ติดลูปตรรกะ หรือออกแบบมาไม่ดีจนพัง วิเคราะห์ว่าเกิดจากอะไรและแก้ยังไง
*   **จุดเด่น:** วิเคราะห์ปัญหาเชิงระบบในกระบวนการ Tool-calling จนนำไปปรับปรุง System Prompt ให้ทนทานขึ้น
